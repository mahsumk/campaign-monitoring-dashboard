# Campaign Monitoring Dashboard (MarTech Simulation)
## Overview
This project simulates a simple internal tool used by marketing teams to monitor campaign performance and detect issues early.
It demonstrates how raw campaign data can be transformed into actionable insights through KPI tracking, automated logic, and clear visualization.
## Problem
Marketing teams need fast visibility into campaign performance across channels and the ability to quickly identify underperforming campaigns.
## Solution
I built a dashboard that:
Tracks key performance indicators (conversion rate, cost per conversion)
Automatically flags campaigns with performance issues
Provides a visual overview of campaign health and efficiency
## Features
KPI calculation (conversion rate, cost per conversion)
Issue detection logic (low conversion, high cost)
Dashboard visualization (performance, efficiency, health)
Summary insights for decision-making
## Tech Stack
Google Sheets (dashboard & visualization)
CSV data processing
## Next Steps
Integrate real-time data via APIs
Automate alerts (Slack/email)
Expand to multi-channel campaign tracking
# Key Takeaway
This project focuses not only on analysis, but on making data usable for decision-making—similar to how internal MarTech tools support campaign managers.
